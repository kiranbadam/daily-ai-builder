# RoboDevLoop Brochure

Landing page for **RoboDevLoop**: "Ship While You Sleep."

## Run locally

```bash
cd brochure
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
cd brochure
npm run build
npm run start
```

## Deploy to Vercel

1. Import this repository into Vercel.
2. Set the **Root Directory** to `brochure`.
3. Keep default framework detection (Next.js).
4. Deploy.

No environment variables are required for the brochure site.
